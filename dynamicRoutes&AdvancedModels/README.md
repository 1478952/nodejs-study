동적 라우팅

- 라우터 경로에 콜론을 추가해 동적 경로 세그먼트를 전달 가능하다
- 콜론 다음에 오는 이름이 req.params 에서 추출할 수 있는 데이터이다.

선택적(쿼리) 매개변수

- URL의 마지막에 물음표를 덧붙인 매개변수로 &으로 나누어 여러개를 가질수 있다.
- req.query 를 통해 값을 추출할 수 있다.
- 라우트에 따로 매개변수를 등록할 필요가 없다
